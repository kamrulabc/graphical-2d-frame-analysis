About Graphical 2D Frame Analysis

Graphical 2D Frame Analysis is free and open-source software for analyzing 2D Frame and portal frames using the stiffness and matrix method. Written in C# and available on GitHub and SourceForge.

Ideal for students and teachers, it provides interactive visualization and detailed numerical Frame Analysis results.

Features – 2D Frame Analysis

  • Axial, shear, bending moment, displacement, absolute displacement diagrams/tables – hover to view values
  • Graphical modeling of beams, columns, supports, loads, and hinges
  • Supports: spring, inclined roller, guided roller, hinge, support displacement
  • Output: all matrices, load vectors, support reactions, global displacements, local forces
  • Open-source C# implementation, Windows OS compatible
