About This Project

Graphical 2D Frame Analysis is a C# (C Sharp)-based educational software designed for students and teachers of civil and structural engineering. It uses the Matrix Method for analyzing multiple types of 2D frame structures, providing graphical and tabular visualization of displacements, axial, shear, and bending moments. Ideal for learning, teaching, and experimentation.

Features

  • First-order linear structural frame graphical matrix analysis
  • Assembly of global matrices for multi-member frames
  • Graphical and tabular visualization of displacements, axial, shear, and bending moment values
  • Supports multiple 2D frame structures with various types of supports:
    • Portal frames
    • Frames with spring supports
    • Frames with roller supports (inclined or guided)
    • Frames with intermediate hinges
    • Frames with absolute displacement constraints
  • Developed in C# (C Sharp) with an intuitive graphical interface
  • Open-source and suitable for civil/structural engineering education
  • Windows OS compatible, easy to experiment and learn
