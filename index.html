About Graphical 2D Frame Analysis

Graphical 2D Frame Analysis is a free and open-source structural engineering software for analyzing 2D frames and portal frames using the stiffness and matrix method. The project is written in C# (C Sharp) and is available with full source code on both GitHub and SourceForge.

The software is ideal for students and teachers in civil and structural engineering. It is designed as an educational, graphical structural analysis tool, providing interactive visualization along with numerical results.

Features

  • Axial force, shear force, bending moment, displacement, absolute displacement diagrams/ graphs and table with values. Any point value of graphs can be shown by moving mouse
  • Draw beams, columns, supports, loads, intermediate hinges graphically
  • Frame with spring support, inclined and guided roller support, intermediate hinges, support displacement
  • Output: All matrices, load vector with indices in addition to support reaction, global displacements, local forces
  • Open-source, C# implementation, Windows OS compatible
